此插件适用于 trilium v0.102.2版本，其他版本未测试。

## 使用方法

1.  从Release下载导入包。
2.  导入到Trilium（取消选择安全导入）。

其中包含两个功能插件，****Table Count and Sum****与****Table Insert/Delete Shortcut Keys****。

---

## Table Count and Sum

1.  鼠标拖拽选中计算区域。
2.  右下角显示计数与求和弹窗。
3.  进行任何取消选中单元格的操作，弹窗消失。

---

## Table Insert/Delete Shortcut Keys

| 快捷键         | 中文说明  |
| ----------- | ----- |
| Ctrl+↑      | 上方插入行 |
| Ctrl+↓      | 下方插入行 |
| Ctrl+←      | 左侧插入列 |
| Ctrl+→      | 右侧插入列 |
| Ctrl+Delete | 删除当前行 |
