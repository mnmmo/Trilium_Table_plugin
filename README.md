This plugin is compatible with Trilium v0.102.2; other versions have not been tested.

## Usage

1.  Download the import package from Releases.
2.  Import it into Trilium (uncheck the "Safe Import" option).

---

## Table Count and Sum

### Usage

1.  Select the first cell in the range to be calculated with the mouse.
2.  Hold down the Shift key, click the last cell in the range to be calculated with the mouse, then release the Shift key.
3.  A popup displaying count and sum will appear in the bottom-right corner.
4.  The popup will disappear when any action that deselects the cells is performed.

There are two scripts here, each corresponding to one of the following two operations. Please keep at most one of them active.

#### Click and drag

1.  Click and drag the mouse to select the calculation range.
2.  A popup displaying count and sum will appear in the bottom-right corner.
3.  The popup will disappear when any action that deselects the cells is performed.

#### shift+click

1.  Select the first cell in the range to be calculated with the mouse.
2.  Hold down the Shift key, click the last cell in the range to be calculated with the mouse, then release the Shift key.
3.  A popup displaying count and sum will appear in the bottom-right corner.
4.  The popup will disappear when any action that deselects the cells is performed.

## Table Insert/Delete Shortcut Keys

| Shortcut Keys | English Description        |
| ------------- | -------------------------- |
| Ctrl+↑        | Insert row above           |
| Ctrl+↓        | Insert row below           |
| Ctrl+←        | Insert column to the left  |
| Ctrl+→        | Insert column to the right |
| Ctrl+Delete   | Delete current row         |
