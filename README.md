This plugin is compatible with Trilium v0.103.0; other versions have not been tested.

## Usage

1.  Download the import package from Releases.
2.  Import it into Trilium (uncheck the "Safe Import" option).

It includes two functional plugins: ****Table Count and Sum**** and ****Table Insert/Delete Shortcut Keys****.

---

## Table Count and Sum

1.  Click and drag the mouse to select the calculation range.
2.  A popup displaying count and sum will appear in the bottom-right corner.
3.  The popup will disappear when any action that deselects the cells is performed.

---

## Table Insert/Delete Shortcut Keys

| Shortcut Keys | English Description        |
| ------------- | -------------------------- |
| Ctrl+↑        | Insert row above           |
| Ctrl+↓        | Insert row below           |
| Ctrl+←        | Insert column to the left  |
| Ctrl+→        | Insert column to the right |
| Ctrl+Delete   | Delete current row         |
